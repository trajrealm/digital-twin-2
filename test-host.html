Digital Twin Widget Test

This is a test page to verify the AI Digital Twin widget embeds correctly and responds to queries with grounded, accurate information.

How to Test

Look for the chat widget in the bottom-right corner of this page. Click to open it and try asking questions about the author. The widget should:

  • Load and display a chat interface
  • Accept your questions
  • Return answers grounded in the author's content (bio, resume, projects, blog)
  • Mark answers as "not confident" if they cannot be reliably answered
  • Offer to escalate low-confidence questions

Example Questions to Try

  • Tell me about the author's background
  • What projects has the author worked on?
  • What are the author's skills?
  • What is the author's work experience?
  • Can you help me with xyz completely unrelated topic?
Note: The widget loads from the local dev server (http://localhost:3000). Make sure the Next.js app is running before testing.
